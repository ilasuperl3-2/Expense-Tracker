# Трекер расходов

## Автор
**Соломин Егор**

## Описание программы
Приложение для отслеживания личных расходов. 

### Основные возможности:
- Добавление расходов с указанием суммы, категории и даты
- Проверка корректности данных
- Фильтрация расходов по категории и диапазону дат
- Подсчёт общей суммы 
- Подсчёт суммы расходов за период
- Редактирование и удаление записей
- Сохранение данных в JSON-файл
- Загрузка данных при запуске программы

## Как использовать

### Установка и запуск
1. Убедитесь, что у вас установлен Python 3.6 или выше
2. Скопируйте файл `main.py` в любую папку
3. Запустите программу командой:
   ```bash
   python main.py
